# ☕ Java Practice Repository

A personal repository where I practice and improve my Java programming skills at an intermediate level.

## About

This repository contains my Java exercises, implementations, and small projects. It is organized by topic to make it easy to navigate and follow my progress.

## 📂 Repository Structure

```
java-practice/
├── algorithms/      # Sorting, searching, recursion, etc.
├── basics/          # Basic Java exercises (variables, loops, OOP, etc.)
├── data-structures/ # Arrays, lists, stacks, queues, trees, etc. 
└── projects/        # Small projects built with Java
```

## Topics Covered

- **Java Basics** — syntax, control flow, object-oriented programming
- **Data Structures** — arrays, linked lists, stacks, queues, hash maps, trees
- **Algorithms** — sorting (bubble, merge, quick), searching, recursion
- **Small Projects** — practical applications to reinforce learning

##  Requirements

- Java 17 or higher
- Any IDE (IntelliJ IDEA, Eclipse, VS Code)

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/YuunoDev/java-practice.git 
   ```
2. Open the project in your preferred IDE.
3. Navigate to any file and run it directly.
